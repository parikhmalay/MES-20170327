The folder contains files for client side application. Mainly includes Angular stuff like Views, Common Resource files for Web Api calls, Controller, Directives etc. /App_Client
  • app.js : Main file for call Angular. Mainly used for app level settings.
  • config.route.js : Used for routes. For load particulars routes & controller call getRoutes() function. Add routes & controlls js on @section scripts { } on MVC view.
  • /views : Contains views,controller & route files per section(Module).
    • /section/viewname.html : View file
    • /section/section.controller.js : controll file
    • /section/section.route.js : route file
    • /_partials : Reuseable controls/pages. For naming convension always start file name with '_'.( like _Loader.html,_Error.html)
  • /common : Used for directives (Shared & View based),helpers,services.
    • /services : data access operation section. file name section.service.js
    • /directives : For common/shared use common.directives.js oterwise create section.directives.js
    • helper.js
